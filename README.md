# Telegram VPN Bot

A Telegram bot that provides VPN service functionality, including:
- Multiple server locations (Russia, America, Germany, Singapore)
- VMess and VLESS link generation
- Account management
- Promo code system
- Referral system

## Setup

### Prerequisites
- Python 3.12.3
- pipenv

### Installation

1. Clone the repository
